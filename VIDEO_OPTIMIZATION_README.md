## Как оптимизировать видео

### Шаг 1: Установите FFmpeg

FFmpeg необходим для оптимизации видео. Установите одним из способов:

**Вариант 1: Через winget (рекомендуется)**
```powershell
winget install ffmpeg
```

**Вариант 2: Через Chocolatey**
```powershell
choco install ffmpeg
```

**Вариант 3: Вручную**
1. Скачайте FFmpeg с https://ffmpeg.org/download.html
2. Распакуйте архив
3. Добавьте папку `bin` в переменную PATH

### Шаг 2: Запустите скрипт оптимизации

Откройте PowerShell в папке проекта и выполните:

```powershell
.\optimize-videos.ps1
```

Скрипт:
- Проверит наличие FFmpeg
- Обработает все видео (кроме LAMBO5.mp4)
- Создаст папку `assets\videos\optimized`
- Конвертирует все видео в MP4 (H.264)
- Оптимизирует для быстрой загрузки в браузере
- Сохранит качество изображения

### Шаг 3: Проверьте результат

После оптимизации:
- Оптимизированные видео будут в `assets\videos\optimized`
- Сайт автоматически использует оптимизированные версии
- Если оптимизированное видео недоступно, используется оригинал (fallback)

## Параметры оптимизации

- **Кодек**: H.264 (libx264) - лучшая совместимость с браузерами
- **Качество**: CRF 23 - баланс между качеством и размером
- **Битрейт**: максимум 2 Mbps для быстрой загрузки
- **Faststart**: включен для начала воспроизведения до полной загрузки
- **Аудио**: AAC 128 kbps, 44.1 kHz

## Примечания

- Оригинальные видео остаются без изменений
- Оптимизированные видео обычно на 30-60% меньше по размеру
- Звук сохраняется во всех видео
- Качество изображения остается высоким

## Если FFmpeg недоступен

Если вы не можете установить FFmpeg сейчас:
- Сайт будет работать с оригинальными видео
- Рекомендуется оптимизировать видео перед публикацией для лучшей производительности
